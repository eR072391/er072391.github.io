CTF Support Page #

このページはCTFを行なう際、役にたつツールなどをまとめています。

Pwn & Rev #

GDB #

GNUデバッガ(GDB)、GNUソフトウェア・システムで動く標準のデバッガ。
多くのUnix系システムで動作可能な移植性の高いデバッガであり、Ada/C言語/C++/Objective-C/Pascal/FORTRAN/FreeBASIC/Goといったプログラミング言語に対応している。

Radare2 #

バイナリをリバースエンジニアリングおよび分析するための完全なフレームワーク。
コマンドラインから独立して使用できる以下の一連の小さなユーティリティで構成されている。

  • [radiff2] バイナリ差分抽出ツール
  • [rabin2] readelf-lineのバイナリ情報抽出ツール
  • [rafind2] バイトパターン検索ツール
  • [rahash2] ハッシュ計算ツール
  • [rarun2] プログラムの実行環境を指定して実行できるツール
  • [rasm2] アセンブラ/逆アセンブラ

retdec #

逆コンパイラ。
ELF/PE/Mach-O/COFF/AR(archive)/Intel HEX/生のマシンコードのファイル形式をサポートし、
Intel x86/ARM/MIPS/PIC32/PowerPCの32 bitのアーキテクチャに対応している。
C言語で記述されたファイルに出力が可能。

Ghidra #

米国国家安全保障局によって開発されたオープンソースリバースエンジニアリングツール。
GitHubで公開されている。Javaで記述されている。

Pwntools #

GallopsledというCTFチームがPwnableを解く際に使用している、Pythonで書かれたCTFフレームワーク。

apktool #

Androidアプリをリバースエンジニアリングするためのツール。

dex2jar #

Androidの.dexやJavaの.classファイル用の解析ツール。

IDA Pro #

逆アセンブラ機能、デバッグ機能、デコンパイラ機能といった数多くの機能を完備したリバースエンジニアリングツール。
逆アセンブラのデファクトスタンダードである。コード間の関連性チェックやAPIコール等を利用してプログラムの自動分析を行なう。

Immunity Debugger #

エクスプロイトの作成、マルウェアの解析、バイナリファイルのリバースエンジニアリングを行なうためのツール。関数のグラフ表示機能を備えたユーザーインターフェイス、ヒープ作成に特化して構築されたヒープ解析ツール。Python APIを基盤に、拡張が可能。

checksec #

実行可能ファイルのPIE/RELRO/Canaries/ASLRなどのセキュリティ機構をチェックするためのbashスクリプト。

rp(++) #

PE/ELF/Mach-Oの実行ファイルおよびx86/x64/ARM/ARM64のアーキテクチャ用のROPガジェットファインダー。

Binary Exploitation Notes #

バイナリエクスプロイトに関しての記述。

libc database search #

シンボルの位置(アドレス)からlibcを調べることができる。

socat #

2つの双方向バイトストリームを確立し、それらの間でデータを転送するコマンドラインベースのユーティリティ。

その他 #

strace Linux用のトレーサの一つ。プログラムが呼び出したシステムコールをトレースする。
ltrace Linux用のトレーサの一つ。プログラムが呼び出した共有ライブラリの関数をトレースする。
file ファイルの種類を調査するコマンド。
readelf LinuxやBSDなどのOSにおける実行ファイルの形式であるELFファイルの情報を表示するコマンド。
strings ファイル中に含まれる表示可能文字列を抽出して表示するコマンド。
grep マッチさせたいパターンとファイルを引数として取り、ファイル中からパターンマッチした行を抽出する。
objdump CUIベースの逆アセンブラ。
xxd ファイルや標準入力から受け取った内容を16進数、または２進数でダンプする。16進ダンプをバイナリに変換が可能。
od ファイルを８進数や１６進数などでダンプする。
hexdump ファイルや標準入力から受け取った内容を８進数や１６進数でダンプする。
printf フォーマット指定子を含むようなフォーマット文字列を利用して、文字列を表示する。
Vim + xxd Vimとxxdコマンドでバイナリファイルを編集する。

Crypto #

FeatherDuster #

脆弱な暗号システムを特定して悪用するプロセスをできるだけ簡単にする暗号解読用のツール。

Hash Extender #

MD4/MD5/RIPEND-160/SHA-0/SHA-1/SHA-256/SHA-512/WHIRLPOOLハッシュをサポートするハッシュ長拡張攻撃を実行するためのユーティリティ。

PkCrack #

既知平文攻撃を使用して、暗号化されているZIPファイルを復号するためのツール。

RsaCtfTool #

RSAマルチ攻撃ツール。脆弱な公開鍵からデータを解読し、秘密鍵の回復を試みる。
特定の公開鍵に対する最適な攻撃の自動選択を行なう。

xortool #

XOR分析を行なうためのツール。

PadBuster #

Padding Oracle攻撃を実行するための自動化するための Perlスクリプト。
任意の暗号文を解読し、任意の平文を暗号化し、自動応答分析を実行し、リクエストがオラクル攻撃のPaddingに対して脆弱かどうかを判断する機能を提供します。

hash-identifier #

ハッシュタイプを識別する。

factordb #

数値の素数DB。RSAを解読する際に活用できるオンラインツール。

hashcat #

Windows/Linux/Macで利用可能なパスワードクラッカー。
総当り攻撃/辞書攻撃/マスク攻撃に対応している。

John The Ripper #

15の異なるプラットフォームで実行可能な、パスワードクラッカー。
パスワードハッシュの種類を自動的に検出することができ、カスタマイズ可能。

dCode #

ゲーム/パズル/コード化されたメッセージ/数字などを解読するのに役立つ800以上のツールセット。
様々な暗号を自動的にデコードが可能なオンラインツール。

Substitution Solver #

単一アルファベット置換暗号の解読オンラインツール。

Vigenere Solver #

Vigenere暗号/Beaufort暗号/Autokey暗号をサポートしている、オンライン解読ツール。

CyberChef #

ブラウザで動作するデータ加工ツール。
Base64やハッシュ値の計算、１６進数への変換と逆変換、QRコードの生成やグラフの描画などが可能。

primefac #

整数を因数分解するためのモジュールおよびコマンドラインユーティリティ。

CrackStation #

事前に計算された大規模なルックアップテーブルを使用して、パスワードハッシュをクラックします。

Forensics #

Aperi’Solve #

画像のレイヤー解析を行なうオンラインプラットフォーム。
zsteg/steghide/outguess/exiftool/binwalk/foremost/stringを使用して、ステガノグラフィ解析を行うことができる。png/jpg/gif/bmp/jpeg/jfif/jpe/tiffをサポートしている。

Exiftool #

画像/音声/ビデオ/PDFメタデータの読み取り、書き込み、操作を行うためのツール。

Binwalk #

ファームウェアイメージの分析、リバースエンジニアリングおよび抽出するためのツール。

Foremost #

ヘッダー/フッター/内部データ構造に基づいて失われたファイルを回復するためのフォレンジックツール。

Steghide #

さまざまな種類のデータを隠すことができるステガノグラフィツール。

StegoVeritas #

自動画像ステガノグラフィ分析ツール。

strong-qr-decoder #

強力なQRコードデコーダ。

QRazyBox #

Webベースで破損したQRコードの分析と回復に使用される。

ImageMagick #

デジタル画像の作成、編集、構成、変換ツール。
PNG/JPEG/GIF/WebP/HEIC/SVG/PDF/DPX/EXR/TIFFなど、２００以上のさまざまな形式の画像をサポートしている。
画像のサイズ変更/反転/ミラーリング/回転/ゆがみ/せん断/変換/画像の色の調整などの描画を行うことができる。

LostMyPass #

クラウドパスワード復元サービス。PDF/MS Office文書/RAR/Zip/7zアーカイブなどのパスワードを解読する。
全て無償ではない。

firmflaws #

ファームウェアをアップロードし、静的分析を実行します。

その他 #

fls ディスクイメージ内のファイルとディレクトリの名前を列挙する。また、inodeで指定されたiノードを使用しているディレクトリの最近削除されたファイル名を表示できる。 icat 指定したiノード番号に基づくファイルのコンテンツを出力する。
dd ブロック単位でファイルをコピー、変換する。

OSINT #

OSINT Framework #

カテゴリー分けされた、様々なOSINT技術が集められたサイト。

CheckUserNames #

500を超えるソーシャルネットワークでユーザー名が使用可能かどうかを確認できる。

BeenVerified #

人物名/電話番号/メール/住所/車両から情報を検索できる。

Shodan #

インターネットに接続されている特定の種類のコンピューターを検索できるようにする検索エンジン。

BuiltWith #

Webサイトがどのようなもので構築されているかを調べられます。

Maltego #

調査対象のドメインやDNS名などからインターネット上の情報を収集し、その関係性を可視化するオープンソースインテリジェンスツール。

theHarvester #

レッドチーム評価または侵入テストの偵察段階で使用するように設計された、使いやすく強力なツール。
ドメインの外部からの脅威の状況を判断するのに役立ちます。
人の名前、電子メール、IP、サブドメイン、URLなどを公開ソースから収集します。

SpiderFoot #

100以上の公開データソースに自動的に問い合わせ、IPアドレス、ドメイン名、メールアドレス、名前などの情報を収集する偵察ツール。

Creepy #

位置情報OSINTツール。オンラインソースから位置情報関連の情報を収集し、地図上に表示したり、正確な場所や日付に基づいてフィルタリング検索をしたり、Googleマップで分析するためにcsv形式またはkmlでエクスポートができる。

Metagoofil #

対象のWebサイトで公開されている公開ドキュメント(pdf、doc、xls、pptなど)のメタデータを抽出するためのツール。ユーザー名、人の名前、共有リソース名、サーバー名などを取得できる。

Raccoon #

whois情報、TLSデータのDNSレコードの取得、サブドメインの列挙、スレッド化されたディレクトリバスティングなど偵察と情報収集に使用されます。

Discover #

DiscoverスクリプトはKali Linuxの偵察ツールをすべて組み合わせたものであり、定期的にメンテナスされています。Discoverスクリプトの受動的なドメインスキャンには、ARIN、DNSRecon、gooflie、goog-mail、Goohost、theHarvester、Metasploit、URLCrazy、whois、複数のWebサイト、それしてRecon-ngが使用されます。

truffleHog #

truffleHogは、さまざまなコミット履歴とブランチでエントロピーの高い文字列をスキャンし、それらを出力するツールです。このため、機密情報、パスワード、キーなどの検出にうってつけです。

SimplyEmail #

SimplyEmailは、domein日時/Email_List.htmlという形式でファイルを出力します。このツールの出力には、企業のメールアドレスのフォーマットと、有効なユーザーのリストが含まれています。SimplyEmailはBing、Yahoo!、Google、Ask Search、PGP Repos、ファイルなどをチェックするため、時間がかかることがあります。

Web #

ffuf #

Webファジングツール。
ディレクトリ/GETパラメータ/HOSTヘッダー/POSTデータのファジングが可能。

Commix #

コマンドインジェクションの脆弱性の検出と悪用を自動化するオープンソースの侵入テストツール。

WPScan #

WordPress Security Scanner。
Rubyで書かれたWordPressの脆弱性スキャンツール。

Webhook.site #

webhookは、web開発でカスタムのコールバックを用いてウェブページやウェブアプリケーションの動作を追加または変更するための方法です。

Posman #

WebAPI開発用ツール。Requestの内容やTokenを設定し、APIに送ることでResponseが返ってきます。

SQLMap #

SQLインジェクションの欠陥を検出して悪用し、データベースサーバーを乗っ取るプロセスを自動化します。

Tplmap #

コードインジェクションおよびサーバーサイドテンプレートインジェクションの脆弱性の悪用を支援し、基盤となるオペレーティングシステムにアクセスするためのさまざまなサンドボックスエスケープ手法を使用します。

XSSer #

WebベースのアプリケーションのXSS脆弱性を検出、悪用、報告するための自動フレームワーク。

SecLists #

セキュリテイアセスメントに役立つ複数種類のデータをリストとしてまとめたもの。
ハッキングの危険性が高いユーザー名/パスワード/URL/機密データなどが含まれたリストです。

git-dumper #

Webサイトからgitリポジトリをダンプするツール。

Web-CTF-Cheatsheet #

Webに関するエクスプロイトコードが集められている。

js-beautifier #

難読化処理されたスクリプトの部分的な難読化解除を行います。

JSFuck #

JavaScriptを、角かっこ([])、丸かっこ(())、プラス記号(+)、感嘆符(!)を使ってエンコードします。JavaScriptの難解な表記を利用できます。

Misc #

Impacket #

ネットワークプロトコルを操作するためのPythonクラスのコレクションです。
Impacketは、パケットへの低レベルのプログラムによるアクセスと、一部のプロトコル(SMB1-3やMSRPCなど)のプロトコル実装自体を提供することに重点を置いています。 NMB/SMB1/SMB2/SMB3/NTLM/Kerberos/MSRPC/TDS/LDAPなどのプロトコルを実装している。

CTF Checklist #

CTF競技でのいくつかのヒント、知識、およびツールについての説明。

REGEXPER #

正規表現の可視化。

PoC-in-GitHub #

GitHubに公開されたCVEのエクスプロイトコードがJSON形式で集められている。

PTF(The PenTesters Framework) #

PTFはスクリプトのコレクションであり、ペネトレーションテストに必要なツール群を自動的にインストールしてくれます。たとえば、エクスプロイトツール、情報収集ツール、ポストエクスプロイトツール、PowerShell、脆弱性分析ツールをすべてインストールできます。

Responder #

Responderは、NTLMv1/NTLMv2/LMv2、拡張セキュリティNTLMSSP、および基本のHTTP認証をサポートする組み込みのHTTP/SMB/MSSQL/FTP/LDAP 不正認証サーバーを備えたLLMNR、NBT-NSおよびMDNSポイズナーです。

dnscat2 #

DNSプロトコル上で暗号化されたC&C(別名C2:コマンドアンドコントロール)チャネルを作成するように設計されています。
